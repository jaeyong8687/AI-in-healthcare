# Synthetic Patient Data

This directory contains synthetic (fictional) patient data for educational purposes in the AI in Healthcare course. **None of this data represents real patients.**

## Files

- `patients_basic.csv` - Basic patient demographics and vital signs
- `patients_diabetes.json` - Detailed diabetes patient records
- `patients_cardiology.json` - Cardiology patient data
- `lab_results.csv` - Sample laboratory test results

## Important Notes

1. **All data is synthetic and generated for educational purposes only**
2. This data is designed to represent realistic clinical scenarios for learning
3. Do not use this data for actual clinical decision-making
4. Data follows HIPAA-compliant synthetic data generation principles

## Usage

These datasets can be used for:
- Machine learning model training exercises
- Data analysis assignments
- Clinical decision support system development
- Healthcare AI ethics discussions
- Data privacy and security training

## Data Privacy

While this data is synthetic, students should still practice proper data handling:
- Treat it as if it were real patient data
- Follow data security best practices
- Understand HIPAA and privacy regulations
- Learn proper de-identification techniques
